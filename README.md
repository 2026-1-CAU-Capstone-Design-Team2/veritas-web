# VERITAS — 제품 랜딩 · 다운로드 사이트

**VERITAS** 는 추론·검색·검증·저장을 모두 내 PC에서 처리하는 **로컬 우선(Local-first) AI 리서치·문서 작성 어시스턴트**입니다.
이 저장소는 VERITAS를 소개하고 Windows 설치 파일(무료) 다운로드로 전환시키는 **정적 마케팅 사이트**입니다.


---


## 빠른 시작

```bash
# 요구사항: Node.js 18.18+ (개발은 Node 22로 검증)
npm install
npm run dev     # http://localhost:3000

```
